# MVP Calculator Cursor

MVP сервис "Просчёт Задачи MVP" для онлайн-специалистов:
- ввод задачи свободным текстом;
- автоопределение категории;
- уточнение деталей;
- расчёт стоимости с коэффициентами;
- fallback без OpenAI/HH.

## Запуск

1. Установить зависимости:
   `npm install`
2. Создать `.env` на основе `.env.example` (опционально добавить `OPENAI_API_KEY`).
3. Запустить:
   `npm run dev`
4. Открыть:
   [http://localhost:3000](http://localhost:3000)
